To-Do Dodo
Android App

This is a Proof-of-Concept Android app written in Java. It's designed with the following concepts:
Publisher-Subscriber Model, Model-View-Viewmodel (MVVM), and Repository Pattern.

This used the following concepts in Android Development:

  • Building with the Publisher-Subscriber Model.
  • Building with the Model-View-Viewmodel.
  • Building with the Repository Pattern.
  • Building and upgrading the database with the Room Library.
  • Implementing Data Binding.
  • Used onSavedInstanceState for Device Orientation.
  • Implemented the alert dialog.
